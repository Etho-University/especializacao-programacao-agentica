# Coding Agents

| Ferramenta | Licença | SWE-bench | Quando escolher |
|---|---|---|---|
| **OpenHands** (All-Hands-AI) | MIT | alto | coding agent open source robusto |
| **OpenDevin** | MIT (renamed → OpenHands) | — | predecessor do OpenHands |
| **Devin** (Cognition) | comercial | comercial alto | coding agent comercial |
| **OpenCode** (sst) | open source | — | CLI coding agent |
| **SWE-agent** | MIT (research) | alto | pesquisa |

## Comparativo

- **Open source robusto**: OpenHands (sucessor de OpenDevin).
- **Comercial líder**: Devin (Cognition).
- **CLI**: OpenCode.
- **Pesquisa**: SWE-agent (base de muitos papers).

## Quando usar qual

- **Dev individual open source**: OpenHands ou OpenCode.
- **Empresa com budget**: Devin.
- **Pesquisa/estudo**: SWE-agent (ler paper, entender arquitetura).

## Nota

OpenDevin foi renomeado para OpenHands em 2024. Citação histórica mantida.
