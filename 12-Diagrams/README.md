# 12 — Diagramas

Material visual da Especialização. Formatos preferenciais: **Mermaid** (versionável em markdown) · **Structurizr/C4** (para arquiteturas) · **PlantUML** (UML) · **PNG/SVG exportados** para slides.

## Convenção

- `ETHAGT0x/` — diagramas do curso
- `_shared/` — diagramas reutilizáveis (agent loop, augmented LLM, etc.)
- Nomear por tema: `agent-loop.mmd`, `supervisor-topology.mmd`, `eval-pipeline.mmd`

## Tipos cobertos

- **C4** (Context · Container · Component · Code) para arquiteturas
- **UML** (sequência, classes, estados) para designs
- **Fluxos** (workflow dos 5 padrões Anthropic)
- **Hierarquias** (topologias multi-agente)
- **Mind maps / timelines** (panoramas)
- **Comparativos** (tabelas visuais, matrizes)
