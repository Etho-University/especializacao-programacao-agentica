# 06 — Projects

Projetos aplicados por módulo. Diferem dos labs por serem **abertos** (menos passo-a-passo, mais resolução de problema realista). Validam o Pilar Prático (10%) e parte do Técnico (40%).

## Convenção

- `ETHAGT0x/projeto.md` — enunciado
- `ETHAGT0x/starter/` — código inicial (opcional)
- `ETHAGT0x/solution/` — solução de referência (restrito a mentores)

## Características esperadas

- Cenário **realista** (simula cliente, domínio, restrições)
- Entregáveis múltiplos: código + docs + eval report
- Critérios de sucesso **mensuráveis**
- Conexão explícita com competências do módulo
