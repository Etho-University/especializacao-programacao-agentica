# 19 — Examples

Exemplos **runnable** de referência. Diferem dos labs por serem **pequenos e focados** em demonstrar um padrão específico — não um tutorial completo.

## Convenção

- `ETHAGT0x/exemplo-NN-[tema]/` — pasta auto-contida
- Cada exemplo: `README.md` (como rodar) + código + `requirements.txt`/`pyproject.toml`

## Princípios

- **Mínimo** — só o necessário para demonstrar o padrão
- **Runnable** — clonar e rodar sem fricção
- **Comentado** — explica o **porquê**, não o **o quê**
- **Sem segredos** — `.env.example` com placeholders
