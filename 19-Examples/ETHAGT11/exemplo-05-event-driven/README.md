# Event-Driven Agents — Exemplo Mínimo

Simula agentes que reagem a eventos via fila in-memory (padrão pub/sub simplificado).

## Como rodar

```bash
pip install pydantic
python main.py
```

## O que demonstra

- Agentes desacoplados via eventos
- Produtores e consumidores independentes
- Escalabilidade horizontal (adicionar consumidores)
