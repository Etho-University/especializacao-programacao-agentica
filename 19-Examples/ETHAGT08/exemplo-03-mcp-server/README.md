# MCP Server — Exemplo Mínimo

Servidor MCP minimalista em FastMCP com duas tools.

## Como rodar

```bash
pip install mcp httpx python-dotenv
python server.py
```

Em outro terminal, conecte com qualquer cliente MCP.

## O que demonstra

- Criação de MCP server com FastMCP
- Tools com schemas ACI bem desenhados
- Retorno estruturado (JSON)
