# 23 — Exams

Provas e gabaritos para o Pilar Técnico (40%). Uma prova por módulo + exame de certificação.

## Convenção

- `ETHAGT0x/prova.md` + `ETHAGT0x/gabarito.md`
- `certification/blueprint.md` — blueprint do exame de certificação
- `certification/exame-vN.md` — versões do exame

## Tipos de questão

- Múltipla escolha (conceito)
- Verdadeiro/falso justificado
- Análise de trade-off (redação curta)
- Análise de arquitetura (dado um diagrama, criticar)
- Debug de agente (dado um trace, encontrar problema)
- Projeto curto (escrever tool / corrigir prompt)
