# 20 — Research (Knowledge Base)

Base de conhecimento estruturada que sustenta toda a Especialização. **Evidence-based**: nenhum conteúdo didático é produzido sem fonte consultada e registrada aqui.

- [`master-index.md`](master-index.md) — **índice master** (v1): todas as fontes por categoria com arXiv, ano, importância, módulos. **Comece aqui.**

- `frameworks/` — análises de frameworks (a preencher)

- Fichas por módulo: `ETHAGT0x-pesquisa.md` (raiz deste diretório) — fontes consultadas por curso.
