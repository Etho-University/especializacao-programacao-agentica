# 04 — Apostilas

Material textual aprofundado de cada curso. **Nível pós-graduação lato sensu** — não superficial. Cada apostila é o material de referência duradouro do curso (os slides são auxiliares).

## Convenção

- Uma pasta por curso: `ETHAGT0x/`
- Estrutura típica:
  - `00-introducao.md` — motivação, panorama, o que você vai aprender
  - `01-fundamentos.md` — teoria profunda
  - `02-padrones.md` — padrões com trade-offs
  - `03-implementacao.md` — mão na massa
  - `04-producao.md` — considerações de produção
  - `05-casos-estudo.md` — casos reais
  - `06-referencias.md` — bibliografia

## Profundidade esperada

Cada apostila deve permitir que um aluno **compreenda o porquê** das decisões (trade-offs, alternativas), não apenas **como** rodar um framework. Justificar com papers, benchmarks e experiência real.
