# Políticas de Uso da Plataforma Etho

## 1. Termos Gerais

Ao utilizar a plataforma Etho, o usuário concorda com os seguintes termos.

## 2. Uso Aceitável

A plataforma deve ser utilizada apenas para fins legais e em conformidade com as leis brasileiras.

## 3. Privacidade

Os dados dos usuários são tratados conforme a LGPD (Lei Geral de Proteção de Dados).

## 4. Responsabilidades

O usuário é responsável pela segurança de suas credenciais de acesso.

## 5. Disponibilidade

A plataforma disponibiliza 99,9% de uptime, excluídas manutenções programadas.

## 6. Suporte

Suporte disponível de segunda a sexta, das 9h às 18h.

## 7. Alterações

A Etho reserva-se o direito de alterar estes termos a qualquer momento.
