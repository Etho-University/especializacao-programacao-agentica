# 05 — Labs

Laboratórios guiados **hands-on**. Cada lab tem objetivo claro, roteiro passo-a-passo, ferramentas e entrega esperada. O princípio pedagógico *Aprender Fazendo* vive aqui.

## Convenção

- `ETHAGT0x/lab-NN-[tema].md` — roteiro
- `ETHAGT0x/lab-NN-[tema]/` — código inicial / solução

## Estrutura de cada lab

1. **Objetivo** (1 frase)
2. **Preparação** (ambiente, deps, dados)
3. **Roteiro** (passos numerados com checkpoints)
4. **Desafios extras** (para quem terminar cedo)
5. **Entrega** (o que subir no repositório)
6. **Rubrica** (link para `08-Assignments/`)
