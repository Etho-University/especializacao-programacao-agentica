# DSPy Optimizer — Exemplo Mínimo

Otimização de pipeline com DSPy (compilação de módulo).

## Como rodar

```bash
pip install dspy openai python-dotenv
echo "OPENAI_API_KEY=sk-..." > .env
python main.py
```

## O que demonstra

- DSPy Module (programa declarativo)
- Compilador BootstrapFewShot (otimização automática)
- Separação programa vs parâmetros
