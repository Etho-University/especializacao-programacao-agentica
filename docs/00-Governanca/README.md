## Por que governar?

Uma Especialização desta escala (~440 h, 16 cursos, múltiplos autores, evolução contínua) **degrada sem governança**. As decisões tomadas aqui (prefixo `ETHAGT`, foco em arquitetura, ênfase em multi-agentes, modelo de 4 pilares, rastreabilidade competência→módulo→avaliação) são **imutáveis por princípio** e só mudam via ADR.
