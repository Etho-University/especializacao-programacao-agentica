# 03 — Slides

Apresentações por curso. Formato: **Markdown** (Marp/Slidev) ou **reveal.js** — versionável, diff-friendly, exportável para PDF/PPTX.

## Convenção

- Uma pasta por curso: `ETHAGT0x/`
- Arquivos: `01-intro.md`, `02-conceito.md`, …, `99-resumo.md`
- Assets em `ETHAGT0x/assets/` (imagens, diagramas)

## Princípios visuais

- Máximo **1 ideia por slide**
- Diagramas > texto (referenciar `12-Diagrams/`)
- Snippets de código curtos com destaque do essencial
- Citar fonte em cada slide com dado empírico
