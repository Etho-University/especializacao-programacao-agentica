# Tool Agent

> Padrão 23/23 · Categoria F — Meta-Agência e Evolução

## Intenção
Encapsular um domínio de ferramentas (ex.: "SAP agent", "GitHub agent").

## Estrutura
Agente especializado num conjunto coeso de tools de um domínio.

## Quando usar
Domínio com muitas tools relacionadas; quer isolamento.

## Anti-patterns
Tool agent fazendo tudo (vira god object).

## Custo
Igual a agente normal.
