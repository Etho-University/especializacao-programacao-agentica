# Memory Manager

> Padrão 15/23 · Categoria D — Informação e Memória

## Intenção
Gerenciar memória do sistema — o que armazenar, recalls, eviction.

## Estrutura
Hooks para armazenar eventos, recuperar relevantes, aplicar política.

## Quando usar
Agentes de longo prazo.

## Anti-patterns
- Armazenar tudo (custo)
- Sem eviction

## Custo
Baixo (gerenciamento).
