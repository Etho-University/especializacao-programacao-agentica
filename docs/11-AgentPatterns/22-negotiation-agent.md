# Negotiation Agent

> Padrão 22/23 · Categoria F — Meta-Agência e Evolução

## Intenção
Negociar entre partes (agentes, recursos) para chegar a acordo.

## Estrutura
Proposta → contraproposta → (loop) → acordo ou impasse.

## Quando usar
Multi-agent com objetivos conflitantes.

## Anti-patterns
- Sem critério de parada
- Deadlock

## Custo
N× chamadas.
