# 08 — Assignments

Trabalhos avaliativos com **rubricas explícitas**. Cada assignment declara quais pilares mede e como.

## Convenção

- `ETHAGT0x/assignment-NN.md` — enunciado + rubrica
- Usa [`24-Templates/template-rubrica-avaliacao.md`](../24-Templates/template-rubrica-avaliacao.md)

## Tipos

- **Técnico**: prova, projeto, implementação
- **Consultivo**: apresentação para "cliente", estudo de caso
- **Comportamental**: code review, colaboração
- **Prático**: demo ao vivo, Q&A
