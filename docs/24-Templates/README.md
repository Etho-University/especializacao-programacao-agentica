# 24 — Templates

Modelos padronizados para produzir material consistente em toda a Especialização. **Sempre copie** o template relevante ao iniciar um novo artefato.

## Templates disponíveis

| Template | Quando usar |
|---|---|
| [`template-syllabus.md`](template-syllabus.md) | Ao criar/emendar um curso `ETHAGT0x` em `02-Syllabus/` |
| [`template-adr.md`](template-adr.md) | Ao registrar uma decisão arquitetural/pedagógica (`00-Governanca/adr/` ou `10-Architecture/adr/`) |
| [`template-rubrica-avaliacao.md`](template-rubrica-avaliacao.md) | Ao definir como um módulo/capstone é avaliado (Pilar Técnico 40%) |
| [`template-eval-report.md`](template-eval-report.md) | Ao relatar resultados de avaliação de um agente/sistema (AgentOps) |
| [`template-pdi-agentic.md`](template-pdi-agentic.md) | PDI específico para alunos da Especialização (estende o Template de PDI Etho) |
