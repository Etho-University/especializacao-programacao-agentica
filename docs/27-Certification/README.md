# 27 — Certification

Critérios, blueprint do exame e processo de certificação da Especialização.

## Critérios de certificação

| # | Critério | Como |
|---|---|---|
| 1 | 16 módulos + capstone aprovados (≥ 3,0) | Sistema |
| 2 | Última avaliação trimestral ≥ **4,0** | Sistema |
| 3 | ≥ 8 sessões de mentoria | Sistema |
| 4 | **Capstone aprovado** (≥ 4,0 por painel) | Campo específico |

**Validade**: 2 anos.

## Recertificação

- Concluir ≥ 2 eletivos (`ETHAGT-E*`), **ou**
- Contribuir com ≥ 1 pattern/arquitetura/caso de estudo mesclados no repositório, **ou**
- Mentorear ≥ 2 alunos até a Fase A.
